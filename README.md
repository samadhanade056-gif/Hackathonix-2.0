# Hackathonix-2.0
 Smart Internship Portal with AI Skill Matching

A web platform that helps students find the most relevant internships using Artificial Intelligence.
The system analyzes student resumes, extracts skills using NLP, and matches them with internship requirements to recommend the best opportunities.
Problem Statement

Many students struggle to find internships that match their skills, interests, and career goals.
